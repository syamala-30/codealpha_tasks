# 🎵 AI Music Generation using LSTM (CodeAlpha Internship - Task 3)

## 📌 Project Overview
This project is an AI-based music generator that learns musical patterns from MIDI files and generates new music using a deep learning model (LSTM - Long Short Term Memory).

The model is trained on classical MIDI datasets and produces a new MIDI file as output.

---

## 🚀 Features
- Loads MIDI dataset files
- Extracts musical notes and chords using music21
- Converts data into training sequences
- Builds and trains an LSTM neural network
- Generates new music sequences
- Converts output into a playable MIDI file

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- music21
- NumPy
