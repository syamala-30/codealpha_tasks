# 🌍 AI Language Translation Tool (CodeAlpha Internship - Task 1)

## 📌 Project Overview
This project is a simple AI-based Language Translation Tool built using Python and Streamlit.  
It allows users to translate text between multiple languages using a clean and interactive web interface.

---

## 🚀 Features
- Multi-language text translation  
- Automatic language detection  
- Text-to-Speech (TTS) output  
- Simple and interactive Streamlit UI  
- Real-time translation results  

---

## 🧠 Technologies Used
- Python  
- Streamlit  
- deep-translator  
- gTTS  
- langdetect  

---

## 📁 Project Structure

CodeAlpha_Language_Translator/
│
├── app.py              # Main Streamlit application
├── requirements.txt    # Dependencies
└── README.md  

---

## ⚙️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

---

### 2. Run the application
streamlit run app.py

---

## 🌐 Output

After running the application, Streamlit will open in your browser at:

http://localhost:8501

### Output Behavior:
- User enters text input  
- Selects source and target languages  
- Clicks Translate button  
- The translated text is displayed instantly on the screen  
- Optional audio output is generated using TTS
