# 🤖 FAQ Chatbot using NLP (CodeAlpha Internship - Task 2)

## 📌 Project Overview
This project is a simple FAQ Chatbot built using Natural Language Processing (NLP).  
It answers user questions by matching them with the most relevant FAQ using TF-IDF vectorization and cosine similarity.

The chatbot provides fast and intelligent responses based on predefined questions and answers.

---

## 🚀 Features
- NLP-based text preprocessing  
- TF-IDF vectorization for text representation  
- Cosine similarity for question matching  
- Intelligent FAQ response system  
- Interactive chat UI using Streamlit  
- Real-time user interaction

## 🎯 How it Works

1. User enters a question  
2. Text is cleaned and preprocessed  
3. TF-IDF converts text into numerical vectors  
4. Cosine similarity finds the closest matching FAQ  
5. The best answer is displayed to the user
